# MindFlow Project fyp-2021/2022

## Project Descriptions

A project that aims towards researcher in aiding drawing mind map while reading paper and jotting down notes.

## Development of Project

Require Node ([Version â‰¥ 18.6.0](https://nodejs.org/en/download/current/)) and Yarn ([Installatinon Guide](https://classic.yarnpkg.com/en/docs/install#windows-stable)).

Install dependency

```bash
yarn install
```

Developing Project

```bash
yarn start
```

Publish the application

```bash
yarn publish
```

Run tests of Project Using Jest

```bash
yarn test
```
